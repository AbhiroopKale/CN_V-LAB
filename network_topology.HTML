Aim

To simulate Star, Bus and Ring network topologies.

Theory

Network topology defines how devices are connected. Star uses a hub, Bus uses a shared cable, Ring uses circular connection.

Procedure

  1. Select topology
  2. Click send packet
  3. Observe transmission

Simulation

Pretest

1. Star uses hub

2. Bus has collision

3. Ring avoids collision

4. Hub failure stops network

5. Bus is cheapest

Posttest

1. Ring avoids collision

2. Bus uses single cable

3. Star depends on hub

4. Ring failure affects all

5. Star scalable
